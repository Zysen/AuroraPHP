{F.EventStream} calmE(time)
Calms this event stream to fire at most once every time ms. Events that occur sooner are delayed to occur time milliseconds after the most recently-fired event. Only the most recent event is delayed. So, if multiple events fire within time, only the last event will be propagated.
Parameters:
{!number|F.Behavior} time
Returns:
{F.EventStream}
{F.EventStream} collectE(init, combine)
Transforms this event stream to produce the result accumulated by combine.

The following example accumulates a list of values with the latest at the head:

original.collectE([],function(new,arr) { return [new].concat(arr); });
